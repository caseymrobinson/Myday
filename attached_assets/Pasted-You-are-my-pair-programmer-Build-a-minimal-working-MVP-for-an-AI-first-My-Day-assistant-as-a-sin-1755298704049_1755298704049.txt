You are my pair-programmer. Build a minimal, working MVP for an AI-first “My Day” assistant as a single Replit project with:

  - /web (React + Vite + TypeScript + Tailwind)

CORE FEATURES (MVP)
1) Calendar ingest (read-only):
   - Fetch events from a Google Calendar *secret iCal URL* (no OAuth).
   - Server cron (node-cron) pulls and caches events every 15 min.
   - Parse with node-ical. Normalize to {id, title, start, end, location, description, isAllDay}.

2) Tasks:
   - Prisma models:
     - Task { id, title, source ('manual'|'slack'|'ai'), status ('pending'|'confirmed'|'done'), priority INT, dueAt DATETIME | null, estimateMins INT | null, url TEXT | null, context JSON, aiSuggested BOOLEAN, createdAt, updatedAt }
     - FocusBlock { id, taskId FK, start DATETIME, end DATETIME, confirmed BOOLEAN }
   - REST endpoints: 
     - GET /api/tasks
     - POST /api/tasks (create)
     - PATCH /api/tasks/:id (update status/confirm/done)
     - POST /api/ingest/slack (stub endpoint: accept JSON { text, url?, ts? } -> extract naive title + create Task with source='slack', aiSuggested=true). No Slack OAuth yet.

3) “What am I doing today?” endpoint:
   - GET /api/agenda?date=YYYY-MM-DD -> returns:
     {
       meetings: [normalized events for that day],
       freeBlocks: [{start, end}],
       topTasks: top 5 tasks (pending + due soon or high priority),
       suggestions: schedule suggestions (see #4) marked AI-only (no writes)
     }

4) Simple scheduler (server-side):
   - Greedy fill freeBlocks with topTasks (estimateMins or default 30m), respecting existing meetings.
   - Return “suggestions” array with proposed FocusBlocks but DO NOT persist unless confirmed.

5) OpenAI hook (keep optional, behind env OPENAI_API_KEY):
   - POST /api/chat { message }:
     - If message matches intent “what am I doing today”, call agenda and return a structured natural-language summary.
     - Otherwise echo stub. Leave room for future LLM parsing of Slack text -> tasks.

WEB UI (/web)
- React + Vite + Tailwind. Use FullCalendar for the calendar.
- Three panes (responsive):
  a) Chat dock (right drawer): input box calling /api/chat and showing responses.
  b) Calendar center: FullCalendar Day/Week view showing:
     - Confirmed meetings (solid color).
     - AI-suggested focus blocks (semi-transparent + dashed border + “AI” pill).
  c) Tasks left panel:
     - Sections: “Unconfirmed (AI)”, “My Tasks”, “Done”.
     - Each task shows title, due chip, source icon (Slack/manual/AI).
     - Buttons: [Confirm] -> status='confirmed', aiSuggested=false and creates FocusBlock; [Done] -> status='done'.
- Toolbar buttons: “Today’s Plan” (calls /api/agenda for today), “Add Task”.
