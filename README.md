# My Day - AI Assistant

An AI-first daily planning assistant that combines calendar integration, intelligent task scheduling, and natural language interaction to help you manage your day effectively.

## Features

- **Calendar Integration**: Sync with Google Calendar via iCal URL
- **Smart Task Management**: Organize tasks by source (manual, Slack, AI) and priority
- **AI-Powered Scheduling**: Get intelligent suggestions for focus blocks
- **Natural Language Chat**: Ask your AI assistant about your schedule and tasks
- **Three-Pane Interface**: Tasks, calendar, and chat in a unified view
- **Automatic Sync**: Calendar updates every 15 minutes
